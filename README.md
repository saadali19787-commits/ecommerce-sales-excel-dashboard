# E-commerce Sales Analysis (Excel Project)

## Project Overview

This project analyzes e-commerce sales data to understand trends, profit, and product performance.

## Tools Used

* Microsoft Excel

## Features

* Pivot Tables
* Pivot Charts
* Interactive Dashboard using Slicers

## Key Insights

* Sales vary by category and region
* Some products generate most of the revenue
* Profit is not equal across all regions
* Seasonal trends affect sales
